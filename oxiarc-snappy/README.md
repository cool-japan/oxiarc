# oxiarc-snappy

Pure Rust Snappy compression library, part of the OxiArc ecosystem.

## Features

- Block format for in-memory compression
- Framed format with CRC32C checksums
- Streaming Write/Read API
- Pure Rust — no C dependencies

## Usage

```toml
[dependencies]
oxiarc-snappy = "0.2.6"
```
